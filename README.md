# KIBAD — Аналитическая Студия

**Профессиональная аналитика данных без программирования**

KIBAD — многостраничное приложение для полного цикла анализа данных. Позволяет специалистам без навыков программирования загружать данные, очищать их, строить визуализации, проводить статистические тесты, прогнозировать временные ряды и генерировать отчёты.

## Возможности

| Категория | Функции |
|-----------|---------|
| **Данные** | CSV, Excel, JSON, PostgreSQL; NoCode SQL с JOIN, GROUP BY, HAVING и вычисляемыми колонками |
| **Подготовка** | Очистка, дедупликация, импутация пропусков, нормализация, пользовательские формулы, IFRS 9, SICR |
| **Анализ** | EDA, корреляции, кластеризация, факторный анализ, оценка качества данных (0–100) |
| **Статистика** | t-тест, Манна-Уитни, ANOVA, хи-квадрат, Bootstrap, Power Analysis |
| **Прогноз** | ARIMA, Prophet, ETS, STL, обнаружение аномалий, ACF/PACF |
| **Моделирование** | Монте-Карло (VaR/CVaR), сценарный анализ, Roll-Rate, Марковские цепи |
| **Визуализация** | 18 типов интерактивных графиков, сравнение периодов |
| **Объединение** | JOIN, UNION, конкатенация с диагностикой ошибок слияния |
| **Автоматизация** | Пайплайны с шаблонами, авто-аналитик, текстовая аналитика |
| **Экспорт** | HTML/PDF отчёты, форматированный Excel, Markdown, аудит-лог |

## Установка

### Системные требования

- Python 3.9 или выше
- Git

### Шаги установки

1. Клонируйте репозиторий:

```bash
git clone https://github.com/stpnmk/kibad.git
cd kibad
```

2. Создайте виртуальное окружение (рекомендуется):

```bash
python -m venv .venv
source .venv/bin/activate  # macOS/Linux
# или
.venv\Scripts\activate     # Windows
```

3. Установите зависимости:

```bash
pip install -r requirements.txt
```

---

## Запуск

```bash
streamlit run app/main.py
```

Приложение откроется в браузере по адресу: **http://localhost:8501**

> **Примечание для macOS:** если функция экспорта в PDF недоступна, установите системную библиотеку:
> ```bash
> brew install pango
> ```

## Страницы приложения

### Начало работы

| Страница | Описание |
|----------|----------|

### Данные и подготовка

| Страница | Описание |
|----------|----------|
| **1. Данные** | Загрузка файлов (CSV, Excel, Parquet), подключение к PostgreSQL, NoCode SQL с JOIN/GROUP BY/HAVING, предпросмотр данных |
| **2. Подготовка** | 9 шагов очистки: типы данных, даты, пропуски, выбросы, нормализация, дедупликация, пользовательские формулы, IFRS 9, SICR |
| **3. Группировка** | Агрегация по группам, сводные таблицы, скользящие статистики |
| **4. Объединение** | JOIN, UNION, конкатенация датасетов с диагностикой pitfalls (дублирование строк, несовпадение типов, низкий процент совпадений) |

### Анализ

| Страница | Описание |
|----------|----------|
| **5. Исследование** | EDA: корреляции, распределения, профиль данных, точечная диаграмма, оценка качества данных 0–100 |
| **6. Тесты** | Автодиагностика данных → рекомендация теста → выполнение: t-тест, Манна-Уитни, хи-квадрат, Bootstrap, Power Analysis |
| **7. Временные ряды** | Декомпозиция STL, прогнозирование (ARIMA, ETS, Prophet), ACF/PACF, обнаружение аномалий |
| **8. Атрибуция** | Факторный анализ: аддитивный, мультипликативный, регрессионный, Shapley Values |
| **9. Моделирование** | Монте-Карло (VaR, CVaR), сценарные шоки, анализ «что если» |
| **12. Кластеризация** | K-means, иерархическая кластеризация, DBSCAN, визуализация кластеров |
| **14. Roll-Rate** | Матрица Roll-Rate, Марковские цепи, анализ переходов между состояниями |

### Визуализация и отчёты

## Структура проекта

```
kibad/
├── app/
│   ├── main.py              # Главная страница и точка входа
│   ├── state.py             # Управление состоянием сессии
│   ├── styles.py            # Дизайн-система (CSS + компоненты)
│   ├── theme.py             # Токены дизайна (цвета, шрифты, градиенты)
│   ├── components/
│   │   └── ux.py            # Переиспользуемые UI-компоненты
│   └── pages/               # 20 страниц приложения
├── core/                    # Бизнес-логика и алгоритмы
│   ├── data.py              # Загрузка данных (CSV, Excel, Parquet, SQL)
│   ├── prepare.py           # Очистка, типизация, импутация
│   ├── aggregate.py         # Агрегация, сводные таблицы
│   ├── explore.py           # EDA, корреляции, профиль данных
│   ├── tests.py             # Статистические тесты
│   ├── models.py            # Временные ряды, прогнозирование
│   ├── attribution.py       # Факторный анализ
│   ├── simulation.py        # Монте-Карло, сценарное моделирование
│   ├── merge.py             # Объединение датасетов с диагностикой
│   ├── triggers.py          # Правила алертов
│   ├── audit.py             # Аудит-лог операций
│   ├── i18n.py              # Локализация (RU/EN)
│   ├── report.py            # Генерация PDF/HTML отчётов
│   └── validate.py          # Валидация схемы и типов
├── services/
│   └── db.py                # Подключение к PostgreSQL
├── tests/                   # Unit-тесты (343 теста)
├── docs/                    # Документация
├── sample_data/             # Тестовые данные (CSV)
├── smoke_test.py            # Быстрая проверка работоспособности
└── requirements.txt         # Зависимости Python
```
