# KIBAD — Analytics Studio

**Профессиональная аналитика данных без Python, SQL и Excel**

KIBAD — многостраничное Streamlit-приложение для полного цикла анализа данных. Позволяет сотрудникам без навыков программирования загружать данные, очищать их, строить визуализации, проводить статистические тесты, прогнозировать и генерировать отчёты.

---

## Возможности

| Категория | Функции |
|-----------|---------|
| **Данные** | CSV, Excel, JSON, PostgreSQL (NoCode SQL с JOIN/GROUP BY/HAVING) |
| **Подготовка** | Очистка, дедупликация, импутация, формулы, IFRS 9, SICR, карточки До/После |
| **Анализ** | EDA, корреляции, кластеризация, факторный анализ, оценка качества данных |
| **Тесты** | t-тест, Манна-Уитни, ANOVA, A/B, Bootstrap, Power Analysis |
| **Прогноз** | ARIMA, Prophet, ETS, Монте-Карло (VaR/CVaR), сценарное моделирование |
| **Визуализация** | 18 типов графиков, KPI-дашборд, сравнение периодов |
| **Автоматизация** | Пайплайны с шаблонами, авто-аналитик, Roll-Rate + Марковские цепи |
| **Экспорт** | HTML/PDF отчёты, форматированный Excel, Markdown |

## Быстрый старт

### Требования

- Python 3.9+
- pip

### Установка

```bash
git clone <repo-url>
cd kibad
pip install -r requirements.txt
```

### Запуск

```bash
streamlit run app/main.py
```

Приложение откроется по адресу `http://localhost:8501`

## Страницы

### 📥 Данные и подготовка
- **1. Данные** — загрузка файлов, PostgreSQL, NoCode SQL с JOIN/HAVING/вычисляемыми колонками
- **2. Подготовка** — 9 шагов: типы, даты, пропуски, выбросы, нормализация, формулы (IFRS 9, SICR)
- **4. Объединение** — JOIN, UNION, конкатенация с диагностикой pitfalls

### 🔬 Анализ
- **5. EDA** — корреляции, распределения, профиль данных, **оценка качества 0–100**
- **6. Тесты** — диагностика данных → рекомендация теста → выполнение + **Power Analysis**
- **7. Временные ряды** — ARIMA, ETS, Prophet, STL, ACF/PACF, аномалии
- **9. Моделирование** — сценарные шоки + **Монте-Карло** (VaR, CVaR)

### 📊 Визуализация и отчёты
- **18. KPI Дашборд** — метрики со светофором RAG, тренды, сравнение периодов
- **20. Графики** — 18 типов (Bar, Line, Scatter, Heatmap, Treemap, Funnel, Свечи, Двойная ось, Анимированный)
- **21. Пайплайн** — визуальный редактор автоматизации, 6 готовых шаблонов
- **24. Шаблоны** — пошаговые сценарии с авто-навигацией и трекером прогресса
- **25. Авто-аналитик** — полный анализ одним кликом: QC, распределения, корреляции, аномалии, рекомендации

## Архитектура

```
Пользователь
    │
    ▼
Streamlit UI (app/pages/)
    │
    ▼
Бизнес-логика (core/)  ←→  Данные (st.session_state)
    │
    ▼
Сервисный слой (services/)  +  Аудит-лог (core/audit.py)
```

**Ключевые принципы:**
- Состояние: все датасеты в `st.session_state`, управляется через `app/state.py`
- Дизайн: токены в `app/theme.py`, CSS в `app/styles.py` (шрифт Inter, градиенты)
- i18n: RU/EN через `core/i18n.py`
- Аудит: все операции логируются в `core/audit.py`
