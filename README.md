# KIBAD — Аналитическая Студия

**Профессиональная аналитика данных без программирования**

KIBAD — многостраничное приложение на Plotly Dash для полного цикла анализа данных. Позволяет специалистам без навыков программирования загружать данные, очищать их, строить визуализации, проводить статистические тесты, прогнозировать временные ряды и генерировать отчёты.

## Возможности

| Категория | Функции |
|-----------|---------|
| **Данные** | CSV, Excel, Parquet, PostgreSQL; каталог датасетов, профилирование |
| **Подготовка** | 9-шаговая очистка: типы, даты, пропуски, выбросы, нормализация, дедупликация, формулы, IFRS 9, SICR |
| **Анализ** | EDA, корреляции, распределения, KPI-трекер, оценка качества данных (0–100) |
| **Статистика** | t-тест, Манна-Уитни, хи-квадрат, Bootstrap, перестановочный тест, A/B-тест, BH-коррекция |
| **Прогноз** | Naive, ARX, SARIMAX; бэктестинг, ACF/PACF, обнаружение аномалий |
| **Моделирование** | Монте-Карло (VaR/CVaR), сценарный анализ, Roll-Rate, Марковские цепи |
| **Визуализация** | 18 типов интерактивных графиков, сравнение периодов, водопадные диаграммы |
| **Объединение** | JOIN, UNION, конкатенация с диагностикой (дупликация, совпадение типов) |
| **Автоматизация** | Пайплайны, авто-аналитик, текстовая аналитика, шаблоны сценариев |
| **Экспорт** | HTML/PDF отчёты, форматированный Excel, аудит-лог |

## Установка

### Системные требования

- Python 3.9 или выше
- Git

### Шаги установки

1. Клонируйте репозиторий:

```bash
git clone https://github.com/stpnmk/kibad.git
cd kibad
```

2. Создайте виртуальное окружение (рекомендуется):

```bash
python -m venv .venv
source .venv/bin/activate  # macOS/Linux
# или
.venv\Scripts\activate     # Windows
```

3. Установите зависимости:

```bash
pip install -r requirements.txt
```

---

## Запуск

```bash
python app/main.py
```

Приложение откроется в браузере по адресу: **http://localhost:8501**

Для продакшена (gunicorn):

```bash
gunicorn app.main:server -b 0.0.0.0:8501
```

python -m pytest tests/ -v

python -m pytest tests/ -v --cov=core --cov-report=term-missing

├── app/

│   ├── aggregate.py           # Агрегация

│   ├── simulation.py          # Монте-Карло

│   ├── cluster.py             # Кластеризация
